## How To Use ğŸ”§

From your command line, first clone E-commerce crwn app âš¡ï¸:

```bash
# Clone this repository
$ git clone https://github.com/Res01uti0n/E-commerce.git

# Go into the repository
$ cd E-commerce

# Remove current origin repository
$ git remote remove origin
```

Then you can install the dependencies either using NPM or Yarn:

Using NPM:
```bash
# Install dependencies
$ npm install

# Start development server
$ npm start
```
Using Yarn:
```bash
# Install dependencies
$ yarn

# Start development server
$ yarn start
```

**NOTE**:
You need also to connect Firebase and Stripe, run frontend.

```sh
cd client

npm/yarn start
```

and type in browser [http://localhost:3000](http://localhost:3000)
